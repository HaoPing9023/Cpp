# C++ 数据结构与算法练习仓库

> 本仓库用于展示我在 C++、数据结构与算法方面的系统自学过程，包含手写 STL 容器、自定义数据结构、算法题解以及面向对象相关示例代码。  
> 代码均由本人独立编写，持续维护。

---

- **核心语言**：C++  
- **主要方向**：数据结构与算法、STL 底层实现、面向对象编程  
- **实践方式**：  
  - 手写实现 `vector / list / stack / queue / priority_queue / string` 等容器  
  - 自己实现二叉搜索树等基础数据结构  
  - 通过算法题巩固时间复杂度、空间复杂度与常用解题思路  
  - 使用小项目和示例代码理解类设计、继承、多态等概念
- **开发环境**：Windows + Visual Studio（可在其他支持 C++11+ 的编译器上编译运行）

---

## 仓库结构说明

> 说明：**名字与 STL 容器相同的文件夹（如 `vector`、`list` 等），都是我在练习“手撕”实现对应容器。**

- `vector/`  
  手写 `std::vector` 的练习代码，包含：
  - 动态数组管理与扩容策略  
  - `push_back / pop_back / operator[]` 等接口  
  - 拷贝构造、赋值重载等基础对象语义

- `list/`  
  手写双向链表，模拟 `std::list` 行为，包括：
  - 结点结构设计  
  - 头插、尾插、任意位置插入 / 删除  
  - 简单迭代器实现

- `stack_queue/`  
  栈与队列的基础实现与练习代码：
  - 顺序表 / 链表实现栈和队列  
  - 基本接口与边界条件处理

- `PriorityQueue/`  
  手写优先级队列（堆）：
  - 大根堆 / 小根堆  
  - 上滤 / 下滤操作  
  - 与 `std::priority_queue` 接口对齐的练习

- `String/`  
  模拟实现 `std::string`：
  - 动态内存管理与深拷贝  
  - 拼接、追加、下标访问等常用操作  
  - 简单查找与比较逻辑

- `BinarySearch/`  
  **二叉搜索树（Binary Search Tree, BST）实现相关代码：**
  - 二叉搜索树结点结构设计  
  - 插入、查找、删除等核心操作  
  - 中序遍历验证有序性  
  - 用模板类封装，支持不同键值类型的 BST

- `Date_Calc/`  
  自定义日期类 `Date`：
  - 日期加减（运算符重载 `+/-/+=/-=`）  
  - 日期比较运算  
  - 计算两个日期之间相差的天数

- `algorithm/`  
  已做过的算法题代码集合：
  - 数组、字符串、链表、树等常见题型  
  - 二分、双指针、前缀和、哈希、基础 DP 等  
  - 用于巩固刷题思路与代码实现能力

- `test_cpp/`  
  C++ 基础语法和特性的小练习：
  - 引用、指针、`const`、内联函数等  
  - 类与对象、运算符重载等语法实验

- `继承/`  
  学习面向对象时编写的继承相关示例：
  - 基类 / 派生类关系  
  - 虚函数与多态  
  - 构造、析构顺序与重写行为
